# Ardum demo video

## What this is

A 60-90 second demo video for the UXmaxx hackathon submission. The video
shows Ardum — an agentic yoga retreat booking platform that starts from
intention, not inventory.

## Key beats

1. **The thesis** (10s) — "Travel booking starts from intention, not inventory"
2. **The live site** (15s) — the practitioner journey: intention → recommendation → hold
3. **The grant ceremony** (10s) — booking with chain abstraction (Magic + Particle UA)
4. **Agent-driven booking** (15s) — an AI agent books a retreat autonomously via the API
5. **On-chain proof** (10s) — the real Arbiscan transaction, verified
6. **The TAM** (10s) — "Any AI agent can use Ardum as booking infrastructure"
